# MUVICO LLM

Good at recommending movies.

## Getting Started

### Usage

`python version 3.11`

#### Using the LLM

```
python src/muvico.py
```

#### Fine Tuning

```
python llm.py
```

## Model

[TinyLlama/TinyLlama-1.1B-Chat-v1.0](https://huggingface.co/TinyLlama/TinyLlama-1.1B-Chat-v1.0)

## Dataset

[MovieLens 1M Dataset](https://grouplens.org/datasets/movielens/1m/)
